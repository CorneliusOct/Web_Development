    Linking element

    To create a link in html to another page or website we use <a> known as an achor element, followed by a href element. ie <a href="https://developer.mozilla.org/en-US/docs/Web/HTML/Element/a">

    This is an achor element for more info on said tag Anchor reference

    When using an achor tag it will open a link in the same window. To open a link in a new tab we add the attribute target="_blank"

    • Further attributes can be added to the anchor tag rel. rel is the relation to the current page and the linked page. For security purposes using rel="noopener noreferrer" prevents the anchored link from referencing the web page and knowing where the link comes from.

    Absolute and relative links

    Absolute links are links that are domains or other websites like https://youtube.com

    Relative links are links to other pages within the website and use the href title of the html file like about.html about page

    Images

    the tag <img> is the tag for images to be displayed.

    like the anchor tag it also needs a reference known as a src= the source can be a link to an image or it could be in house using a directory.

    The image tag has an attribute that has a description of an image when it can not be loaded. alt="description here"

    The image tag has another attribute for its display size height="" and width="" setting only one of the attributes will scale the image accordingly.
